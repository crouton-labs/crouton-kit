# Verify Clean

Run quality gates to ensure code is ready for commit.

1. **Typecheck**: Run compiler checks
2. **Lint**: Run ESLint to catch code quality issues
3. **Test**: Run test suite to verify functionality - unit tests and integration tests

Run all checks. Be mindful of long outputs so tail appropriately.
